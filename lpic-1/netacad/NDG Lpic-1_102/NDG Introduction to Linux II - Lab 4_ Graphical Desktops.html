4.1 Step 1

For the first portion of this lab, you will be using the CentOS PC virtual machine.

Select the CentOS PC by clicking the CentOS PC tab in the virtual machine window on the right side of the screen.

In this lab you will be alternating between the Gnome and KDE desktops and displaying the memory usage for each.

First, login to the root account. When prompted for the password, use the root password netlab123:

CentOS Linux 7 (Core)
Kernel 3.10.0-1062.el7.x86_64 on an x86_64

centos login: root
Password:
[root@centos ~]#

4.2 Step 2

Next, switch to the GUI by bringing the system to runlevel 5 using the init 5 command:

init 5
[root@centos ~]# init 5

4.3 Step 3

Click the sysadmin user. Now, login as sysadmin using the password netlab123 and press the Enter key or click Sign In

The desktop environment may take a minute to load.

4.4 Step 4

In the desktop environment, click the Applications button and select Terminal from the drop-down menu.

4.5 Step 5

The GUI is built upon several layers, a display server, a display manager, a window manager, and a desktop environment.

In the terminal, use the pstree command to view which display manager the GUI is using:

pstree | grep dm

According to the output, this GUI is using gdm (Gnome Display Manager).

4.6 Step 6

To view which GUI your system is currently using, enter the following command into the terminal:

env

The env command will display the environment variables on the system. In the output of the env command, the XDG_CURRENT_DESKTOP=GNOME-Classic:GNOME variable means that this system is currently using the Gnome desktop.

4.7 Step 7

To view the memory usage for the Gnome desktop, use the following command:

free -h

4.8 Step 8

For this portion of the lab, you will be using the Ubuntu PC virtual machine.

Select the Ubuntu PC by clicking the Ubuntu PC tab in the virtual machine window on the right side of the screen.

Click the sysadmin user. Now, login as sysadmin using the password netlab123 and press the Enter key (or click Login).

The desktop environment may take a minute to load.

4.9 Step 9

In the desktop environment, click the Applications icon at the bottom left corner of the screen, then select the Favorites tab and lastly, click the Type to search bar at the top of the System menu.

4.10 Step 10

In the search bar, type the following:

Konsole

Konsole is the terminal for the KDE graphical user interface. Once the Konsole application appears in the search results, click the Konsole application:

4.11 Step 11

In the terminal, use the pstree command to view which display manager the GUI is using:

pstree | grep dm
sysadmin@ubuntu:~# pstree | grep dm

According to the output, this GUI is using sddm (Simple Desktop Display Manager).

4.12 Step 12

To view which GUI your system is currently using, enter the following command into the terminal:

env
sysadmin@ubuntu:~#  env

In the output of the env command, the XDG_CURRENT_DESKTOP=KDE variable means that this system is currently using the KDE desktop.

4.13 Step 13

To view the memory usage for the Gnome desktop, use the following command:

free -h
sysadmin@ubuntu:~# free -h

Based on the output of the free command, the used memory is 371. Recall, that the output of the free command on the CentOS system in Step 7 shows that the memory used on the CentOS system was 512. Based on this comparison, it appears that the KDE desktop on this Ubuntu system is less resource intensive than the Gnome desktop on the CentOS system used at the beginning of the lab.

This lab is complete.
