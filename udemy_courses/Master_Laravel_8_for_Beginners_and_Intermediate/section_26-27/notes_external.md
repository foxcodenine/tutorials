### How to Get Logged in User Data in Laravel?
https://www.itsolutionstuff.com/post/how-to-get-logged-in-user-data-in-laravel-5example.html

Get Logged User Data using helper:

    $user = auth()->user();
    print_r($user);

Get Logged User Data using facade

    $user = Auth::user();
    print_r($user);
