Install create-nuxt-app Globaly

    $ sudo install -g create-nuxt-app

To create a new nuxt project: 
    $ create-nuxt-app project_name

To use scss:
    $ npm install --save-dev node-sass sass-loader

To install axios: 
    $ npm install --save axios
